# 背景圖片

可替換為自己的圖片，檔名對應如下（或到 `src/config/backgrounds.ts` 修改路徑）：

| 檔名 | 使用位置 |
|------|----------|
| `hero-bg.svg` 或 `hero-bg.jpg` | 首屏 Hero |
| `section-bg-1.svg` 或 `.jpg` | 二、服務範疇與核心價值 |
| `section-bg-2.svg` 或 `.jpg` | 四、專案備註與責任界定 |

建議：使用淺色、低對比圖片，或搭配 `src/style.css` 中 `.section-bg-overlay` 的透明度，以維持文字可讀性。
